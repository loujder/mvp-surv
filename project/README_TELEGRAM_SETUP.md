
## Настройка

### 1. Создание Telegram Bot

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен бота
3. Настройте Web App через команду `/newapp`

### 2. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
# Telegram Bot Token
TELEGRAM_BOT_TOKEN=your_bot_token_here

# Admin Chat IDs (comma-separated)
ADMIN_CHAT_IDS=508246426

# Telegram Web App Secret
TELEGRAM_WEBAPP_SECRET=your_webapp_secret_here

# Environment
NODE_ENV=development
```

### 3. Настройка базы данных

База данных будет автоматически обновлена при первом запуске. Новые поля будут добавлены к существующей таблице User.

### 4. Развертывание

1. Убедитесь, что все переменные окружения настроены
2. Перезапустите backend и frontend
3. Откройте приложение через Telegram Web App

## Администраторы

Администраторы определяются по `chat_id` в переменной `ADMIN_CHAT_IDS`. Текущий администратор: `508246426`.

Для добавления новых администраторов добавьте их `chat_id` в список через запятую:

```env
ADMIN_CHAT_IDS=508246426,123456789,987654321
```

## Безопасность

- Все данные от Telegram проверяются на подлинность через HMAC-SHA256
- `chat_id` является уникальным и не может быть подделан
- Администраторы определяются только по `chat_id` из конфигурации
